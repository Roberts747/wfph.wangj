# JCOG0802临床试验分析

## 项目简介

本项目是基于JCOG0802/WJOG4607L临床试验的完整分析，专注于早期非小细胞肺癌手术策略研究。由潍坊市人民医院胸外科医学中心王江整理，为医学专业人士提供循证医学指导。

## 主要内容

### 🏠 首页 (index.html)
- 研究背景与临床问题
- JCOG0802试验概述
- 核心临床问题分析
- 肺结节与磨玻璃结节手术指南

### 📊 详细结果 (results.html)
- 5年和10年生存数据分析
- 交互式图表展示
- 亚组分析结果
- 并发症与安全性数据

### 🔬 研究方法 (methodology.html)
- 研究设计详解
- 入选与排除标准
- 随机化方法
- 统计学设计

## 核心发现

JCOG0802试验证实：对于≤2cm的周围型浸润型NSCLC，肺段切除术在总生存方面优于肺叶切除术，为早期肺癌手术策略提供了重要循证依据。
