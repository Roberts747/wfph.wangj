入选与排除标准

严格的入组条件确保研究质量

主要入选条件

基本特征

  • • 年龄20-85岁
  • • ECOG PS 0-1
  • • 预计术后FEV1≥800mL
  • • PaO₂≥65 torr

影像学特征

  • • 单发周围型结节
  • • 位于肺实质外1/3
  • • 肿瘤最大径≤2cm
  • • 薄层CT测得CTR>0.5
  • • 影像学N0

排除标准

肿瘤位置

  • • 右中叶病灶
  • • 靠近肺门
  • • 无法保证切缘

患者因素

  • • 既往同侧开胸史
  • • 既往恶性肿瘤放化疗史
  • • 器官功能不全

术中二次确认

术中需快速病理证实NSCLC，排除恶性胸腔积液或胸膜种植， 系统淋巴结探查证实无N1-2转移，技术上确认段切或叶切均可行。

手术方案

标准化的手术操作要求

叶切组

  • 标准解剖性叶切除
  • 肺门淋巴结清扫
  • 纵隔系统淋巴结清扫
  • 确保完整切除

段切组

  • 解剖性肺段或双肺段切除
  • 肺门淋巴结清扫
  • 纵隔系统淋巴结清扫
  • 切缘≥肿瘤径或≥20mm

段切组特殊要求

不允许的术式
  • • 广泛基底段切（相当于叶切）
  • • 非解剖性楔形切除
切缘阳性处理
  • • 术中冰冻提示切缘阳性需追加切除
  • • 无法保证切缘时可转为叶切

研究终点

主要终点和关键次要终点设定

主要终点

总生存 (OS)

  • • 基于ITT人群分析
  • • 从随机分组到全因死亡
  • • 非劣效设计，HR界值1.54
  • • 允许优效性检验

次要终点

肿瘤相关

  • • 无复发生存 (RFS)
  • • 局部复发率
  • • 复发模式分析

功能相关

  • • 术后6月、12月FEV1
  • • 肺功能变化

安全性

  • • 围手术期并发症
  • • 死亡率
  • • 住院时长
